# 🚀 AI Resume Analyzer

An AI-powered web application that analyzes resumes, calculates ATS scores, and provides actionable insights to improve job readiness.

---

## 🌐 Live Demo
[Live App](https://ai-resume-analyzer-jthekp3dbymhzzggnu8kci.streamlit.app)  

---

## 💻 GitHub Repository
https://github.com/arpitjaiswal-26/AI-RESUME-ANALYZER  

---

## 🔍 Features

- Resume analysis using AI  
- ATS score calculation  
- Job description matching  
- Skill detection & missing skills  
- Strengths & weaknesses analysis  
- PDF upload + text input  
- Login system  
- History tracking  

---

## 🛠 Tech Stack

- Python  
- Streamlit  
- Groq API (LLaMA 3)  
- SQLite  
- PyPDF2  
- bcrypt
